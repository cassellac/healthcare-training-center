Presentation Design Guide

  • Title slides: Navy-to-teal gradient, white text, centered, 56px+ headlines
  • Room showcase slides: Light surface background, room color accent, icon feature, description
  • Content slides: White background, navy headings, teal bullets, 18px+ body text
  • Closing slide: Full-color gradient, centered, motivational copy
  • Spacing: 60px padding all sides, generous whitespace between sections
  • Fonts: Montserrat for all headings (600–800 weight), Inter for body (400–600 weight)
  • Colors: Navy (#1E3A6E) primary, Teal (#2B8FA9) accent, room colors for highlight
  • Imagery: Phosphor icons, warm healthcare photography, no busy textures
  • Slide numbers: Bottom right, small (12px), gray (#94A3B8)
  • Consistency: Keep the footer (date, URL, slide number) visible on all slides

Document Design Guide

  • Header: Colored top border (navy or room color), company name, document title, meta info
  • Body copy: 16px Inter, 1.6 line height, 600px max line width (readability)
  • Headings: Montserrat (700–800 weight), navy color, 8px grid spacing
  • Lists: Teal bullet/arrow, 18px left margin, 8px line spacing
  • Emphasis: Bold navy text (not italics) for key points
  • Boxes/callouts: Light surface background, teal left border, 20px padding
  • Page breaks: 96px padding before/after major sections
  • Footer: Page number (bottom right), document version (bottom center), date
  • Margins: 40px all sides, 8px grid-aligned
  • Colors: Navy headings, gray body copy (#475569), teal accents, surface backgrounds

Export & Sharing Tips

  • For presentations: Export as PDF (preserves formatting), or import HTML into PowerPoint/Google Slides
  • For documents: Save as PDF for final distribution, or export as Word doc for further edits
  • For sharing: Use PDF for read-only sharing (looks consistent on all devices)
  • File naming: Use kebab-case: "healthcare-training-center-pitch-deck-june-2026.pdf"
  • Accessibility: Always add alt text to images; use proper heading hierarchy (H1 → H2 → H3)
  • Print: Test print layout; ensure color-safe versions exist for monochrome printing
