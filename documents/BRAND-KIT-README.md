## 🎯 Using These Files in AI Tools (Canva, Adobe Firefly, etc.)

**Best practices for uploading to AI design tools:**

1. **Screenshots work better than PDFs**
   - Open HTML files in browser
   - Screenshot color swatches, typography samples, components
   - Upload images to your design tool

2. **Copy text references to your prompts**
   - Paste color codes (`#1E3A6E`, `#2B8FA9`) directly into tool prompts
   - Include typography specs ("Montserrat Bold 36px for headings")
   - Use voice guidelines from `SKILL.md`

3. **Use the templates as starting points**
   - Download/screenshot template designs
   - Import into Canva, Adobe, Figma as reference
   - Build on top of the provided structure

4. **Icon library reference**
   - Share `ICON-LIBRARY.json` in your prompt
   - Tell the AI: "Use Phosphor icons named: ph-users, ph-shield-check, etc."
   - Most AI tools recognize Phosphor by name

5. **Keep links handy**
   - If these files are hosted (on Pages or GitHub), share the URL
   - Most AI tools can read HTML directly from a link
   - Example: "Reference colors from [your-url]/BRAND-GUIDE.html"

## 🆘 Troubleshooting

### "The HTML file won't open in my browser"
- Make sure you're opening the file with a web browser (Chrome, Safari, Firefox, Edge)
- Don't try to open with Word or another text editor
- On Mac: Right-click file → Open With → Choose your browser

### "I want to edit a template"
- Open the HTML file in a text editor (VS Code, Notepad++)
- Find the section you want to change (search for text or color code)
- Edit and save
- Open the updated file in your browser to see changes

### "What's the exact color code for Room 2?"
- Look at `SKILL.md` or `BRAND-GUIDE.html`
- Room 2 (HIPAA & Privacy) = `#2BA89A`
- Or check `ICON-LIBRARY.json` for complete color mapping

### "Can I use these templates in [specific tool]?"
- **Canva:** Screenshot the templates, import images, customize
- **Adobe CC:** Copy HTML, paste into Adobe XD or Illustrator, or screenshot and import
- **Figma:** Import HTML as reference, recreate components using design system
- **PowerPoint/Google Slides:** Copy HTML code, paste as "paste special" → Web, or import images

### "I need to update the brand guide"
- Edit `BRAND-GUIDE.html` directly (it's a single self-contained file)
- Or request an update from Claude Code (the AI that built this)
